# Simulador de Automatos Finitos e Gramaticas Formais
Este trabalho, desenvolvido para a disciplina de Teoria da Computação, implementa um software educacional que demonstra conceitos da Teoria da Computação através da criação e manipulação de Autômatos Finitos (AFD e AFND), conversão entre eles, minimização, e operações com Gramáticas Formais, oferecendo uma ferramenta interativa para visualização passo a passo dos algoritmos.

## Sobre o projeto 
O projeto é um software educacional em Python que implementa conceitos fundamentais da Teoria da Computação. Seus principais objetivos são permitir a definição interativa de Autômatos Finitos (AFD e AFND), converter AFND para AFD, minimizar AFDs, testar palavras nos autômatos, definir e classificar gramáticas formais (regulares ou livres de contexto), converter gramáticas regulares para AFD, simplificar gramáticas livres de contexto, derivar palavras a partir da gramática e gerar pseudocódigo de reconhecedores. O programa oferece um menu interativo via terminal, possibilitando explorar todas essas funcionalidades de forma prática e didática.

## Como executar 
1. tenha uma IDE de python instalada e atualizada na sua máquina.
2. execute o programa.
